以下の条件をすべて満たす
「特定商取引法に基づく表記ページ」を作成してください。

# 【目的】
- Stripeの本番審査を通過できる
- 日本の特定商取引法の要件を満たす
- 既存ECサイト（Node.js + EJS + CSS）と
  デザイン・UXの品質を揃える
- ユーザーに不安を与えない、信頼性の高いページにする

# 【前提条件】
- サイト名：セッツマルシェ（City on Farm）
- ビジネス形態：個人開発のマーケットプレイス型EC
- 商品：農作物（単一出品者注文）
- 決済方法：Stripe クレジットカード決済、代金引換
- 返金ポリシー：
  - 不良品・配送事故 → 出品者負担で返金
  - 購入者都合 → 原則不可
  - 送料は返金対象、Stripe手数料は返金対象外
- URLは誰でも閲覧可能（ログイン不要）

# 【実装要件】
## 1. ページ仕様
- URL想定： /tokusho
- EJSテンプレートで実装
- 共通レイアウト（header / footer / meta / SEO）は他ページと同一
- title / meta description / OGP を適切に設定

## 2. デザイン要件（重要）
- 「法務ページだから適当」は絶対にNG
- checkout / about / terms ページと同等のクオリティ
- 可読性・余白・視線誘導を意識
- モバイルファースト
- 情報はカード or セクション分割
- 見出し階層（h1 → h2 → dl / table）を適切に使用

## 3. 記載必須項目（特定商取引法）
以下を **日本の法的表現として自然に** 記載してください。

- 販売事業者名
- 運営責任者
- 所在地
  - ※請求があった場合に遅滞なく開示する表現を使用
- 連絡先（メールアドレス）
- 電話番号
  - ※請求があった場合に遅滞なく開示する表現を使用
- 販売価格（商品ページに税込表示している旨）
- 商品代金以外の必要料金（送料等）
- 支払い方法
- 支払い時期
- 商品の引き渡し時期
- 返品・返金に関する条件
- 販売数量の制限（ある場合）
- その他注意事項（天候・収穫状況等）

## 4. 実装
以下をすべて実装してください。

① EJSテンプレート（views/pages/tokusho.ejs 想定）
② ページ専用CSS（styles/tokusho.css）
③ SEOメタ情報（title / description）
④ フッターリンク文言案
⑤ Stripe審査でそのまま提出できるURL想定文言

# 【重要な制約】
- 法律用語は正確だが、一般ユーザーにも分かる日本語にする
- 不安を煽る表現は禁止
- コピペ感のあるテンプレ文は禁止
- 「あとで決める」「未定」などの表現は禁止
