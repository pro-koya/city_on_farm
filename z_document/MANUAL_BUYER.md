# セッツマルシェ 購入者向けマニュアル

このマニュアルでは、セッツマルシェでの商品購入に必要な操作方法を説明します。

---

# 1. ユーザー登録

## 1.1 サインアップページへアクセス

1. セッツマルシェのトップページ（https://settsu-marche.onrender.com）にアクセス
2. 画面右上の「**新規登録**」ボタンをクリック
3. サインアップページ（/signup）が表示されます

---

## 1.2 アカウント種別の選択

登録フォームの最初に、アカウント種別を選択します。

| 種別 | 説明 |
|------|------|
| **法人（飲食店・事業者）** | 飲食店や事業者として登録する場合。取引先情報の入力が必要です。 |
| **個人** | 個人として登録する場合。取引先情報は不要です。 |

---

## 1.3 必要情報の入力

### 全員共通の入力項目

| 項目 | 入力内容 | 備考 |
|------|---------|------|
| **姓** | お名前（姓） | 例：山田 |
| **名** | お名前（名） | 例：太郎 |
| **メールアドレス** | 有効なメールアドレス | ログインに使用します |
| **パスワード** | 8文字以上推奨 | 英大文字・小文字・数字・記号の組み合わせを推奨 |
| **パスワード（確認）** | パスワードをもう一度入力 | 上と同じ内容を入力 |

### パスワードの強度について

入力したパスワードの強度がリアルタイムで表示されます。

- **弱い**: 短すぎる、または単純すぎます
- **普通**: 基本的な要件を満たしています
- **強い**: セキュリティ的に十分な強度です
- **非常に強い**: 最も安全なパスワードです

### 法人の場合の追加入力項目

| 項目 | 入力内容 | 備考 |
|------|---------|------|
| **取引先名** | 店舗名または法人名 | 例：やまなか食堂 |
| **種別** | 飲食店/小売/卸/その他 | 任意 |
| **電話番号** | 連絡先電話番号 | 例：03-xxxx-xxxx |
| **郵便番号** | 7桁の郵便番号 | 入力すると住所が自動入力されます |
| **都道府県** | 自動入力 | 郵便番号から自動で入力されます |
| **市区町村** | 自動入力 | 郵便番号から自動で入力されます |
| **番地** | 番地を入力 | 例：1-1-1 |
| **建物名・部屋番号** | ビル名など | 任意 |

---

## 1.4 利用規約への同意

1. 「**利用規約**」と「**プライバシーポリシー**」のリンクをクリックして内容を確認
2. 同意する場合、チェックボックスにチェックを入れる
3. チェックを入れると「**登録**」ボタンが有効になります

---

## 1.5 登録完了

1. 「**登録**」ボタンをクリック
2. 入力内容に問題がなければ、認証メールが送信されます
3. 「認証待ち」画面が表示されます

---

## 1.6 メール認証

1. 登録したメールアドレスに認証メールが届きます
2. メール内の「**認証リンク**」をクリック
3. メール認証が完了し、ログイン可能になります

### メールが届かない場合

- 迷惑メールフォルダを確認してください
- 認証待ち画面から「**認証メールを再送**」ボタンをクリックしてください

---

## 1.7 ログイン

1. トップページ右上の「**ログイン**」をクリック
2. 登録したメールアドレスとパスワードを入力
3. 「**ログイン**」ボタンをクリック
4. ダッシュボードにログインできます

### パスワードを忘れた場合

1. ログイン画面で「**パスワードをお忘れですか？**」をクリック
2. 登録したメールアドレスを入力
3. パスワードリセット用のメールが届きます
4. メール内のリンクから新しいパスワードを設定

---

# 2. 多要素認証（MFA）の設定

アカウントのセキュリティを強化するために、多要素認証を設定できます。
認証アプリまたはパスキー（生体認証）の2種類から選択できます。

---

## 2.1 認証アプリ（TOTP）での設定

### 対応アプリ

- Google Authenticator
- Authy
- 1Password
- Microsoft Authenticator

### 設定手順

1. ログイン後、ダッシュボードからアカウント設定へ移動
2. 「**セキュリティ**」セクションの「**二要素認証を設定する**」をクリック
3. 二要素認証設定ページ（/account/2fa/setup）が開きます

### QRコードのスキャン

1. 画面に表示されたQRコードを認証アプリでスキャン
2. 認証アプリに6桁のコードが表示されます

### 確認コードの入力

1. 認証アプリに表示された6桁のコードを入力欄に入力
2. 「**二要素認証を有効化**」ボタンをクリック

### バックアップコードの保存

設定完了後、10個のバックアップコードが表示されます。

**重要**: これらのコードは必ず安全な場所に保存してください。

- 「**すべてコピー**」ボタンでクリップボードにコピー
- 「**ダウンロード**」ボタンでテキストファイルとして保存
- 各コードは1回のみ使用可能です
- 再表示することはできません

「**保存しました - 完了**」ボタンをクリックして設定を完了します。

---

## 2.2 パスキー（生体認証）での設定

パスキーを使用すると、指紋認証や顔認証でログインできます。

### 特徴

| 特徴 | 説明 |
|------|------|
| 高速認証 | ワンタッチで数秒で完了 |
| 高セキュリティ | フィッシング耐性あり |
| デバイス対応 | iPhone/Android/PC対応 |

### 設定手順

1. 二要素認証設定ページで「**生体認証を設定 →**」をクリック
2. 生体認証・パスキー設定ページ（/account/webauthn/setup）が開きます

### デバイスの追加

1. 「**+ 新しいデバイスを追加**」ボタンをクリック
2. デバイス名を入力（例：iPhone 14、MacBook Pro）
3. 「**生体認証で登録**」ボタンをクリック
4. ブラウザの認証画面が表示されます（指紋認証、顔認証など）
5. 認証を実行すると登録完了

### 登録済みデバイスの管理

- **名前変更**: デバイス横の「✏️ 名前変更」ボタン
- **削除**: デバイス横の「🗑️ 削除」ボタン

### ヒント

複数のデバイスを登録しておくことで、デバイスを紛失した際も安心です。

---

## 2.3 ログイン時の多要素認証

### 認証アプリでのログイン

1. メールアドレスとパスワードでログイン
2. 2要素認証画面が表示されます
3. 認証アプリに表示された6桁のコードを入力
4. 「**確認**」ボタンをクリック

### パスキーでのログイン

1. ログイン画面で「**パスキーでログイン**」を選択
2. ブラウザの認証画面が表示されます
3. 指紋認証や顔認証を実行
4. ログイン完了

### バックアップコードの使用

認証アプリを使用できない場合：

1. 2要素認証画面で「**バックアップコードを使用**」を選択
2. 保存しておいたバックアップコードのいずれかを入力
3. ログイン完了（使用したコードは無効になります）

# 3. 商品購入

## 3.1 商品を探す

### トップページから

1. トップページに表示されている商品一覧から選ぶ
2. 気になる商品をクリックして詳細ページへ

### 商品一覧ページから

1. ナビゲーションから「**商品一覧**」をクリック
2. 商品一覧ページ（/products）が表示されます

### 検索・絞り込み

- **キーワード検索**: 検索ボックスに商品名やキーワードを入力
- **カテゴリ絞り込み**: カテゴリを選択して絞り込み

---

## 3.2 商品詳細を確認

商品詳細ページで以下の情報を確認できます：

| 項目 | 説明 |
|------|------|
| 商品名 | 商品の名称 |
| 価格 | 税抜価格 |
| 単位 | 1個あたりの内容量（例：1玉、500g、1箱） |
| 商品説明 | 栽培方法、味の特徴、保存方法など |
| 仕様・規格 | サイズ、等級などの詳細 |
| 出品者情報 | 生産者・出品者の情報 |
| 配送方法 | 常温便、クール便など |
| 発送目安 | 注文から発送までの日数 |

---

## 3.3 カートに追加

1. 商品詳細ページで購入したい数量を選択
2. 「**カートに追加**」ボタンをクリック
3. カートに商品が追加されます

---

## 3.4 カートを確認

1. 画面右上のカートアイコンをクリック
2. カートページ（/cart）が表示されます

### カート内での操作

| 操作 | 方法 |
|------|------|
| 数量変更 | 数量入力欄で変更 |
| 商品削除 | 「削除」ボタンをクリック |
| 小計確認 | 各商品の小計と合計が表示されます |

---

## 3.5 チェックアウト

1. カートページで「**購入手続きへ**」ボタンをクリック
2. チェックアウトページ（/checkout）が表示されます

### ログインしていない場合

チェックアウトに進む前にログインが必要です。
ログイン画面が表示されたら、メールアドレスとパスワードでログインしてください。

---

## 3.6 受け取り方法の選択

出品者の設定により、以下の受け取り方法が選択できます：

### 配送

商品をご指定の住所へ配送します。

1. 「**受け取り方法**」で「配送」を選択
2. 配送先住所を選択または新規追加
3. お届け希望日時を選択（任意）

### 畑で受け取り

出品者指定の畑まで受け取りに行きます。

1. 「**受け取り方法**」で「畑で受け取り」を選択
2. 受け取り希望日時を選択
3. 受け取り場所の詳細は注文確定後のメールで案内されます

**注意**: 当日の受け取りを希望される場合は、直接お電話でご相談ください。

---

## 3.7 配送先住所の入力

### 登録済みの住所から選択

過去に登録した住所がある場合、一覧から選択できます。

### 新しい住所を追加

1. 「**+ 新しい住所を追加**」ボタンをクリック
2. 住所追加モーダルが表示されます
3. 以下の情報を入力：

| 項目 | 入力内容 |
|------|---------|
| お名前 | 届け先のお名前 |
| 電話番号 | 連絡先電話番号 |
| 郵便番号 | 7桁の郵便番号 |
| 都道府県 | 自動入力 |
| 市区町村 | 自動入力 |
| 番地 | 番地を入力 |
| 建物名 | ビル名・部屋番号（任意） |

4. 「**保存**」ボタンをクリック

---

## 3.8 請求先住所

### 配送先と同じ場合

「**配送先と同じにする**」にチェックを入れます。

### 別の住所を指定する場合

1. チェックを外す
2. 請求先住所を選択または新規追加

---

## 3.9 お支払い方法の選択

出品者の設定により、利用可能な支払い方法が表示されます。

### クレジットカード決済

Stripeを通じた安全なカード決済です。

1. 「**クレジットカード**」を選択
2. 確認画面でカード情報を入力

### 代金引換（COD）

商品到着時に配送員へ現金でお支払いいただきます。

1. 「**代金引換**」を選択

---

## 3.10 クーポンの適用

クーポンコードをお持ちの場合：

1. 「**クーポンコード**」欄にコードを入力
2. 「**適用**」ボタンをクリック
3. 割引が適用され、合計金額が更新されます

クーポンを解除する場合は「**解除**」ボタンをクリックします。

---

## 3.11 ご要望・備考

配送に関するご要望がある場合は入力できます。

例：
- 不在時は宅配ボックスへ
- 午前中は不在です

---

## 3.12 注文内容の確認

1. 「**利用規約とキャンセルポリシーに同意します**」にチェック
2. 注文内容を確認：
   - 商品・数量・価格
   - 送料
   - 割引（クーポン適用時）
   - 合計金額
3. 「**確認へ進む**」ボタンをクリック

---

## 3.13 最終確認と決済

確認ページ（/checkout/confirm）で最終確認を行います。

### クレジットカード決済の場合

1. カード情報を入力：
   - カード番号
   - 有効期限
   - セキュリティコード
2. 「**注文を確定する**」ボタンをクリック

### 代金引換の場合

1. 内容を確認
2. 「**注文を確定する**」ボタンをクリック

---

## 3.14 注文完了

1. 注文完了ページ（/checkout/complete）が表示されます
2. **注文番号**が発行されます（大切に保管してください）
3. 登録メールアドレスに確認メールが届きます

---

## 3.15 注文履歴の確認

### 注文一覧

1. ダッシュボードから「**注文履歴**」を選択
2. 最近の注文一覧（/orders/recent）が表示されます

### 注文詳細

1. 注文一覧から確認したい注文をクリック
2. 注文詳細ページ（/orders/:no）が表示されます
3. 以下の情報を確認できます：
   - 注文ステータス
   - 支払い状況
   - 配送状況
   - 商品明細
   - 配送先情報

### 請求書の確認

注文詳細ページから「**請求書（PDF）**」をダウンロードできます。

---

# 困ったときは

2. お問い合わせフォームに内容を入力
3. 「**送信**」ボタンをクリック

担当者より折り返しご連絡いたします。
